#### Couleur, Code Hexa, Usage Recommandé
- Vert Sauge, #7D8F69, "Titres, boutons principaux, icônes ""nature""."
- Terre Cuite, #B56547, "Accents, boutons d'appel à l'action (Réservation)."
- Bois Clair, #D9C5A0, "Fonds de sections, encadrés de texte."
- Blanc Cassé, #F9F7F2, Fond principal du site (plus doux qu'un blanc pur).
- Anthracite Doux, #3E3E3E ,Texte principal pour une lecture reposante.

#### Polices
- Pour les titres : "Playfair Display"

    Style : Serif (avec empattements).

    Effet : Élégant, intemporel et très "salon de thé".

- Pour le corps de texte : "Montserrat" ou "Lato"

    Style : Sans-serif (moderne).

    Effet : Très lisible, propre et accueillant.
